## Description

Back-end development for a tourism company using Node.js, Express & MongoDB. <br>
Front-end development using Pug Templates & CSS styling.

## Usage Information

Publicy deployed at: https://natours-nickderaj.herokuapp.com/ <br>
You can create user accounts, update user information, buy & view the tours that you've booked. <br>
Note: Some functionality like reviews & deleting bookings are not implemented on the front-end.
